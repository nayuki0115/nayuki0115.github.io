# Repo 初始上 Git 步驟

###### tags: `Git`

## 初始化這個目錄，讓 Git 對這個目錄開始進行版控
```script=
git init
```

## 加 README.md (可加可不加)
```script=
git add README.md
git commit -m "first commit"
git remote add origin <URL>
git push -u origin master
```

## 看local有無修改
```script=
git status
```

## 將所有檔案都加入
```script=
git add
```

## 指定放 Git 哪
```script=
git remote add orgin <URL>
```

## 將 code 放上去
```script=
git push origin master
```
