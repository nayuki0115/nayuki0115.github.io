# Git 分支系列指令

###### tags: `Git`

## 新增分支
```script=
git branch <XXXX>
git checkout -b <XXXX>
```

## 切換分支
```script=
git checkout <XXXX>
```

## 刪除分支
```script=
git branch -d <XXXX>
```
